# Options P&L Visualizer

Interactive profit/loss chart for options trades. Visualize risk before you trade.

## Live Demo

https://hcukkai.github.io/options-pl-visualizer

## Features

- Supports calls and puts
- Long and short strategies
- Real-time P&L chart with breakeven line
- Max profit/loss calculation
- Strike price visualization
- Dark theme for trading setups
- Zero data collection - runs entirely in browser

## Use Cases

- Pre-trade risk assessment
- Teaching options mechanics
- Comparing strategies side-by-side
